   Expectations about input:
 1.argv[0] should be the path of the torrent file in the format C:\\Directory1\\Directory2\\cs352.png.torrent

 -If the users' desired file location is not the default, please enter the wanted destination in the format C:\\Directory3\\Directory4\\printerpic.png
